# 課題　3 --メモ帳アプリ

## ①課題内容（どんな作品か）
- トランプゲームのハイアンドロー。ローカルストレージに、ゲームの終了時間や勝敗を保存し履歴として表示させた。

## ②工夫した点・こだわった点
- キーには重複しないように、ゲーム終了時間をわりあてた点
- 前回の課題で心残りだったアラート表示にスタイルをあてた点(SweetAlert2を用いた)
- ゲームの履歴を1個ずつ削除できるようにした点

## ③難しかった点・次回トライしたいこと(又は機能)
- ローカルストレージを使うアイデアが浮かびづらかった。
- より多くのデータをローカルストレージで扱うために、JSON形式に配列を用いて保存まで実装したかった（残件）
